# EShop (Amazon ecommerce site clone)
Eshop is a ecommerce website where people can buy product. It comes with the features like Cart, Payment, Searching, Authentication.
User can search product and add them in the cart.

### How to run the app
1. Download or clone the app and open the folder in cli.
2. Install all dependency using "npm install" command.
3. To start the app use "npm start" . App should run on 'localhost:3000'

### How to use the app
1. In Home page or search any product then click on "Add to cart" to add the product in Cart.
2. In Cart page click on 'procedd to payment' .
3. In payment page fill adress and card info and click pay to buy the product.

### Features
* Cart-
    1. User can add product to Cart without being logged in.
    2. Product can be removed from the cart.
    3. Product number can be increased or decreased.
* Search-
   1. Product can be found by searching using the search bar.
   2. Search keyword is not case sensitive.
 * Payment-
   1. User have to be logged in to buy product.
   2. User address and card info has be to fill out before processing payment
  * Admin-
   1. Admin can upload product and also view all his products.
