# EShop (Amazon ecommerce site clone)
Eshop is a ecommerce website where people can buy product. It comes with the features like Cart, Payment, Searching, Authentication.
User can search product and add them in the cart.

### How to run the app
1. Download or clone the app and open the folder in cli.
2. Install all dependency using "npm install" command.
3. To start the app use "npm start" . App should run on 'localhost:3000'
